# Serapion Server

ExpressJS starter project with JWT authentication implementation and roles.

Install dependencies:
```$xslt
npm install
```

## For development

```$xslt
npm run start
```

## For production
```$xslt
npm run build
npm run serve
```

Server is accessible at http://localhost:3001

## API Documentation:
http://localhost:3001/api/v1/docs
